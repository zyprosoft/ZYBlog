#### 博客包含功能及技术结构
1. 博客文章管理，包括增加、更新、删除等
2. 文章评论管理，包含增加、更新、删除、快捷回复等
3. 简易的分栏管理，可以自定义创建分栏
4. 简易的标签管理，每篇文章可以有最多五个标签
5. 博主个性化信息设置，可以设置个人的头像，QQ扫码二维码，微信扫码二维码，github主页等信息
